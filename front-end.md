- HTML  
    - tags
        - head
            - meta
            - title
            - link
            - style
        - body
            - div
            - h1
            - p
            - a
            - img
- CSS 
    - Responsivo
        - Mobile First
        - Unidades de medida: rem / em
    - Box Model
        - tamanho do conteúdo (width / height)
        - preenchimentos
        - bordas
        - espaçamentos
        - displays
        - cores
            - fundo
            - fonte
            - sombras
            - opacidade
    - Layout
        - posicionamento
            - displays

        - tipo
        - espaçamento
        - alinhamento
